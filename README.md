# PathForge ⚡
### AI-powered career roadmap app for engineering students

Built with Flutter, Firebase & Gemini AI

## 🎯 Problem it solves
Most engineering students don't know which skills to learn, in what order, and where to learn from. PathForge solves this by generating a personalised week-by-week roadmap using AI.

## ✨ Features
- 🤖 AI generated roadmaps using Gemini 2.0
- 📊 4 career tracks: Data Science, AI/ML, Data Engineering, Analytics  
- 🔐 Firebase Authentication (Email + Google)
- 💾 Firestore real-time database
- 📚 Curated YouTube resources (Indian + Global)
- 📱 Cross-platform Flutter app (Android + iOS + Web)
- 🎨 Beautiful purple UI with smooth animations

## 🛠️ Tech Stack
| Technology | Purpose |
|------------|---------|
| Flutter & Dart | Cross-platform mobile app |
| Firebase Auth | User authentication |
| Cloud Firestore | Database |
| Gemini AI API | Roadmap generation |
| Riverpod | State management |
| GoRouter | Navigation |

## 📱 Screens
- Splash Screen
- Auth Screen (Login + Signup)
- Onboarding Quiz (5 questions)
- Track Selection (4 career tracks)
- AI Generating Screen
- Roadmap Screen (week-by-week flowchart)
- Resources Screen (YouTube channels)
- Profile Screen
